Cache methods

## Installation

Add this line to your application's Gemfile:

    gem 'caching'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install caching

## Usage

    class Model
      extend Caching
      
      cache :slow_method, :other_slow_method

      def slow_method
        ...
      end

      def other_slow_method
        ...
      end

      def fast_method
        ...
      end
    end
